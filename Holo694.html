ด้วยศาสตร์ระดับโลก

แอปพลิเคชันพยากรณ์ที่แม่นยำที่สุด ผสาน 10 ศาสตร์ระดับโลก ทั้งดาราโหราศาสตร์ พลังงานตัวเลข และการเสี่ยงทาย ประมวลผลลึกซึ้งถึงระดับจิตวิญญาณ

เลือกศาสตร์การพยากรณ์

จินนี่รวบรวมศาสตร์จากทั่วทุกมุมโลกมาให้เจ้านายเลือกแล้วค่ะ ชอบแบบไหนจิ้มเลย!

โหมดส่วนตัว: เชื่อมโยงจักรวาลโดย AI

สำหรับเจ้าของไฟล์ที่มี API Key ส่วนตัวเท่านั้น หากไม่เปิดใช้งาน คนทั่วไปก็ยังดูคำพยากรณ์ปกติได้เหมือนเดิมค่ะ

แนะนำให้ใช้ไฟล์นี้กับเครื่องของตัวเอง และอย่าแจกไฟล์พร้อม API Key ให้คนอื่นค่ะ

ประกาศความเป็นส่วนตัวของโปรแกรม

โปรแกรมนี้ไม่ได้เก็บข้อมูลส่วนตัวของผู้ใช้ไว้ใน server ของผู้พัฒนา และโหมดพยากรณ์ปกติจะประมวลผลโดยตรงบนเครื่องของผู้ใช้เป็นหลักค่ะ

กรณีที่คุณเปิด โหมด AI พร้อม API Key ข้อมูลสรุปที่จำเป็นสำหรับการตีความจะถูกส่ง ตรงจากเบราว์เซอร์ของคุณไปยัง Gemini API เท่านั้น ไม่ผ่าน server ของผู้พัฒนาไฟล์นี้ค่ะ

ข้อมูลและผลวิเคราะห์จะอยู่เฉพาะใน session การใช้งานนี้ และจะหายไปอัตโนมัติเมื่อ รีเฟรชหน้า ปิดหน้า หรือกดเริ่มใหม่ (ยกเว้นกรณีที่คุณเลือกให้จำ API Key ไว้ในเบราว์เซอร์นี้เอง)

ความเป็นส่วนตัวของการพยากรณ์

โปรแกรมนี้ไม่ได้เก็บข้อมูลของผู้ใช้ไว้ใน server ของผู้พัฒนา และโหมดปกติจะวิเคราะห์ผลโดยตรงบนเครื่องของผู้ใช้เองค่ะ

ถ้าเปิดโหมด AI พร้อม API Key ระบบจะส่งเฉพาะข้อมูลสรุปที่จำเป็น ตรงจากเบราว์เซอร์ของผู้ใช้ไปยัง Gemini API เพื่อขอมุมมองเสริมจาก AI โดยไม่ผ่าน server ของผู้พัฒนา

ข้อมูลและผลลัพธ์จะอยู่เฉพาะใน session นี้ และจะหายไปอัตโนมัติเมื่อ รีเฟรชหน้า ปิดหน้า หรือกดเริ่มใหม่ (ยกเว้น API Key ที่ผู้ใช้เลือกให้จำไว้เอง)
