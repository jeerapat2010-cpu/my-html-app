ขอบเขตระบบ
  • รับเอกสารโครงการวิจัยเพื่อเสนอคณะกรรมการ EC
  • ออกเลขรับ ตรวจ Checklist และพิมพ์ใบรับเอกสาร
  • จัดรอบเข้าพิจารณา EC และบันทึกความเห็นกรรมการ
  • ติดตามเรื่องแก้ไข ส่งกลับ ส่งใหม่ และตรวจซ้ำ
  • ค้นหาแบบ Smart Search พร้อมเส้นทางกระบวนการ
  • ติดตามภาระงานรายเจ้าหน้าที่แบบ Dashboard
ข้อควรทราบ
  • ข้อมูลเก็บใน Browser/เครื่องที่ใช้งาน ควร Backup JSON เป็นระยะ
  • หากเปลี่ยนเครื่องหรือเปลี่ยน Browser ให้ Restore จากไฟล์ Backup
  • การลบชุดเรื่องต้องยืนยันรหัส เพื่อกันกดพลาดระดับหนึ่ง
  • เหมาะสำหรับต้นแบบใช้งานภายในก่อนพัฒนาระบบฐานข้อมูลกลาง
